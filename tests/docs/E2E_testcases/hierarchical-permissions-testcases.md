## 📋 测试概览

分层权限系统测试涵盖权限授予、题库创建、审核流程等核心功能。

### 权限层级

| 层级 | 名称 | 共享范围 | 审核要求 |
|------|------|----------|----------|
| **测评题库** | Assessment | 全市统一 | 严格审核 |
| **市级练习题库** | Municipal | 全市共享 | 市级审核 |
| **区级练习题库** | District | 区内共享 | 区级审核 |
| **校级练习题库** | School | 校内使用 | 无需审核 |

### 测试模块

- **PRM (Permission Management)** - 权限管理
- **QBC (Question Bank Creation)** - 题库创建
- **REV (Review)** - 审核流程

## 🐛 已知问题与修复

### 1. Ant Design 5 表单组件交互问题

**问题**: Select 下拉框和按钮点击超时

**根本原因**:
- Ant Design 5 虚拟滚动导致元素标记为 hidden
- 按钮文本中间有空格（"保 存"、"确 定"）
- 表单字段选择器不一致

**修复方案**:
```typescript
// ❌ 错误
await page.locator('label:has-text("难度")').click();
await page.locator('button:has-text("保存")').click();

// ✅ 正确
await page.locator('.ant-form-item:has-text("难度") .ant-select').first().click();
await page.locator('button').filter({ hasText: /保\s*存/ }).click();
```

### 2. 题目级别字段理解错误

**问题**: 测试尝试选择"校级"/"市级"但字段实际是难度等级 (L1-L9)

**根本原因**: 误解了"题目级别"含义

**修复方案**:
- "题目级别"选择难度等级: L1-基础入门, L2-基础, L3-基础提高, ...
- "发布范围"选择行政级别: 校级题库, 市级练习题库, ...

### 3. 动态选项缺失

**问题**: 测试尝试填写4个选项但只有2个存在

**修复方案**: 点击"添加选项"按钮两次

```typescript
const addOptionButton = page.locator('button').filter({ hasText: /添加选项/ });
await addOptionButton.click(); // 添加选项C
await addOptionButton.click(); // 添加选项D
```
