## 📂 回归测试

### DFT101 - 草稿箱列表显示

**功能描述**: 题库草稿箱能正常显示草稿列表和表格结构

**测试步骤**:
1. 创建一个草稿题目（判断题: 测试草稿箱功能 - 1+1=2）
2. 访问题库主页面 /teacher/question-bank
3. 点击"我的草稿" tab
4. 等待页面加载完成

**验证点**:
1. 表格包含关键列头: 题型、科目、年级、题目内容
2. 表格至少有一行数据（data-row-key）
3. 表格行数 > 0

**优先级**: P0
**状态**: ✅ 通过
**备注**: 基础功能测试通过 | 使用attached状态检查避免虚拟滚动问题

---

### DFT102 - 编辑草稿题目

**功能描述**: 验证能编辑草稿箱中的题目并保存修改

**测试步骤**:
1. 使用`createDraftQuestion`辅助函数创建草稿题目（物理-八年级: 测试编辑功能 - 原始内容）并获取题目编码（questionCode）
2. 进入题库主页面并切换到草稿箱 tab
3. 等待表格加载
4. 使用题目编码精确定位目标行（filter hasText questionCode），点击该行编辑按钮（JavaScript点击）
5. 修改题目内容为"修改后的内容"
6. 提交修改

**验证点**:
1. 点击编辑后进入"编辑题目" 卡片标题页面
2. 表单正确加载原有数据
3. 提交后显示成功消息 (.ant-message-success)

**优先级**: P0
**状态**: ✅ 通过
**备注**: 使用题目编码提升测试稳定性和准确性 | 使用evaluate()避免虚拟滚动可见性问题

**更新日期**: 2025-01-20
**修复内容**:
- 实现题目编码系统（格式：学科代码+日期+序号，如PHYS2510200012）
- 使用 `createDraftQuestion` 辅助函数返回题目编码
- 通过编码精确定位表格行，替代内容文本匹配
- 修复编辑页面预期检查（从"新建题目"tab改为"编辑题目"卡片标题）

### DFT103 - 删除草稿题目

**功能描述**: 验证能从草稿箱删除不需要的题目

**测试步骤**:
1. 创建专门用于删除的草稿（化学-九年级）
2. 进入草稿箱 tab
3. 记录删除前表格行数
4. 点击第一行删除按钮（JavaScript点击）
5. 在确认对话框中点击"确定"
6. 等待操作完成并刷新页面

**验证点**:
1. 删除按钮可正常点击
2. 确认对话框正确显示
3. 删除后显示成功消息
4. 原有表格行数-1，删除成功后行数减少

**优先级**: P1
**状态**: ✅ 通过
**备注**: 使用Promise.race处理短暂的成功消息，测试通过

**更新日期**: 2025-01-20
**修复内容**:
- 使用 `Promise.race` 优雅处理短暂消息显示
- 测试稳定通过
