## 📋 测试要点

### 1. 权限控制
- **练习活动**: 教师和所有管理员可以创建
- **测评活动**: 仅区级及以上管理员可以创建 (district_admin, base_school_admin, municipal_school_admin, municipal_admin)

### 2. 状态流转
- **草稿 (draft)** → 可编辑、可发布、可删除
- **已发布 (published)** → 可取消发布、不可编辑、不可删除
- **进行中 (ongoing)** → 不可取消发布、不可编辑、不可删除
- **已结束 (finished)** → 不可修改

### 3. 能力等级
- L1 - 基础
- L2 - 初级
- L3 - 中级
- L4 - 中高级
- L5 - 高级
- L6 - 优秀
- L7 - 卓越

### 4. 重做机制
- `allowRetake`: 是否允许重做
- `maxAttempts`: 最大尝试次数（1-10次）
