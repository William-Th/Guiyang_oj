## 🔥 冒烟测试

快速验证管理员端核心功能是否正常工作。

### ADM001 - 管理员首页显示测试

**功能描述**: 验证管理员登录后首页元素正常显示

**测试步骤**:
1. 管理员登录
2. 进入首页

**验证点**:
1. 主布局可见
2. 页面头部可见
3. 内容区域可见
4. 管理员功能菜单可见

**优先级**: P0
**状态**: ✅ 通过
**备注**: 验证管理员端UI基本结构

## 👨‍💼 回归测试

管理员和教师的后台管理功能详细测试。

### ADM101 - 管理后台访问

**功能描述**: 教师/管理员能访问管理后台

**测试步骤**:
1. 教师或管理员登录
2. 访问路径 /admin

**验证点**:
1. URL包含 /admin
2. 后台仪表板卡片或表格可见

**优先级**: P0
**状态**: ✅ 通过
**备注**: 后台访问权限
