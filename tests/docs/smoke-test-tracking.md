## 📝 说明

### 什么是冒烟测试？

冒烟测试（Smoke Testing）是软件测试中的一种基本测试类型，用于验证软件的核心功能是否正常工作。名称来源于硬件测试中的"冒烟测试"：如果设备通电后冒烟，说明存在严重问题。

### 测试策略

1. **快速验证**: 测试应在5分钟内完成
2. **核心功能**: 只测试最关键的功能路径
3. **部署后必跑**: 每次部署后都应运行冒烟测试
4. **失败即停**: 任何一个冒烟测试失败都应阻止后续测试

### 运行冒烟测试

```bash
# 运行所有冒烟测试
npm run test:smoke

# 运行特定模块的冒烟测试
npx playwright test smoke --grep "AUT"  # 认证模块
npx playwright test smoke --grep "STU"  # 学生功能模块
npx playwright test smoke --grep "ADM"  # 管理员功能模块
npx playwright test smoke --grep "ACT"  # 活动管理模块

# 运行特定测试用例
npx playwright test smoke --grep "AUT001"
```
