## 测试流程

### 阶段1: 题目创建 (QBC101)

**操作人**: teacher01

**步骤**:
1. 登录系统
2. 进入"题库管理" → "题库管理"
3. 点击"创建题目"
4. 填写题目信息:
   - 题目类型: 单选题
   - 科目: 数学
   - 年级: 二年级
   - 难度: 中等
   - 题目内容: `【完整流程测试】1 + 1 = ?`
   - 选项: A.1, B.2, C.3, D.4
   - 正确答案: B
   - 解析: 1加1等于2
5. 保存为草稿

**预期结果**:
- 题目保存成功
- 状态显示为"草稿"
- 在草稿箱中可见

---

### 阶段2: 提交审核 (R401)

**操作人**: teacher01

**步骤**:
1. 在草稿箱中找到刚创建的题目
2. 点击"提交审核"
3. 选择审核权限类型: `assessment_review` (测评题库审核)
4. 选择审核人A: teacher_yy_ps_01
5. 选择审核人B: teacher_yy_ps_02
6. 确认提交

**预期结果**:
- 提交成功
- 题目状态变为"待审核"
- 题目从草稿箱消失，出现在审核工作台

---

### 阶段3: 第一审核 (R405 - 批准)

**操作人**: teacher_yy_ps_01

**步骤**:
1. 登录系统
2. 进入"审核工作台"
3. 找到待审核题目
4. 查看题目详情
5. 点击"通过"
6. 填写审核意见: "题目内容正确，通过"
7. 确认

**预期结果**:
- 审核成功
- 题目状态变为"第一审核通过"
- 题目流转到第二审核人

---

### 阶段4: 第二审核 (R405 - 批准)

**操作人**: teacher_yy_ps_02

**步骤**:
1. 登录系统
2. 进入"审核工作台"
3. 找到待审核题目
4. 查看题目详情
5. 点击"通过"
6. 填写审核意见: "同意第一审核人意见"
7. 确认

**预期结果**:
- 审核成功
- 题目状态变为"审核通过"
- 题目出现在"已发布"题库中

---

### 阶段5: 创建活动 (ACT101)

**操作人**: admin

**步骤**:
1. 登录系统
2. 进入"活动管理" → "创建活动"
3. 填写活动信息:
   - 活动类型: 练习 (practice)
   - 活动标题: `【完整流程测试】测试活动`
   - 科目: 数学
   - 年级: 二年级
   - 时间限制: 无限制
   - 总分: 100
   - 及格分: 60
   - 范围: 系统
4. 保存草稿

**预期结果**:
- 活动创建成功
- 状态为"草稿"
- paper_status = "empty"

---

### 阶段6: 组卷 (ACT103)

**操作人**: admin

**步骤**:
1. 进入活动详情页
2. 点击"组卷"或"题目管理"
3. 在题库筛选中搜索: `【完整流程测试】`
4. 找到刚审核通过的题目
5. 点击"添加"或选中后添加到试卷
6. 设置题目分值: 10分
7. 保存组卷结果

**预期结果**:
- 题目添加成功
- paper_status 变为 "completed"
- 题目数量更新为1

---

### 阶段7: 发布活动 (ACT104)

**操作人**: admin

**步骤**:
1. 返回活动列表
2. 找到测试活动
3. 点击"发布"
