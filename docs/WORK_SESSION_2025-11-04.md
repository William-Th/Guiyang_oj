#### 2.2 问题诊断与修复

**问题 1: 状态码不匹配**
- 位置: `Admin can grant municipal review permission`
- 现象: 期望201，实际返回200
- 修复:
  ```javascript
  assert(
    response.statusCode === 200 || response.statusCode === 201,
    'Should return 200 or 201'
  );
  ```

**问题 2: 缺少 target_scope 参数**
- 位置: `Complete workflow` 测试 Step 3
- 现象: 立即发布时缺少 target_scope 参数
- 修复: 添加 `target_scope: 'practice_municipal'`

**问题 3: 字段名错误**
- 位置: 审核API调用
- 现象: 使用 `comments`（复数），应为 `comment`（单数）
- 修复: 修改字段名

**问题 4: 审核人权限不匹配** (关键问题)
- 位置: `Complete workflow` 测试 Step 2-3
- 现象: `"You are not the assigned reviewer for this question"` (403)
- 原因:
  - Step 2 中 reviewer_id = testData.teacherId (94号用户)
  - Step 3 中使用 testData.teacherToken (teacher01)
  - 两者不是同一用户
- 修复:
  ```javascript
  // Step 2: 将审核人设置为管理员
  reviewer_id: 1, // 使用管理员作为审核人（ID=1）

  // Step 3: 使用管理员token审核
  Authorization: `Bearer ${testData.adminToken}`
  ```
