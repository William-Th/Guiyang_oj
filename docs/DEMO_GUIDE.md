## 角色层级体系

```
Level 5: 系统总管理员 (admin)
   └─ Level 4: 市级管理员 (guiyang_admin, municipal_school_admin, base_school_admin)
        └─ Level 3: 区县级管理员 (yunyan_admin, nanming_admin, ...)
             └─ Level 2: 校级管理员 (school_admin_yy_ps_01, ...)
                  └─ Level 1: 教师 (teacher_yy_ps_math, ...)
```

**账号创建规则**:
- 每一级管理员可以创建下一级管理员的账号
- 系统总管理员可以创建所有级别的账号
- 市级管理员可以创建区县、校级、教师账号
- 区县级管理员可以创建校级、教师账号
- 校级管理员可以创建教师账号
- 教师不能创建其他账号
