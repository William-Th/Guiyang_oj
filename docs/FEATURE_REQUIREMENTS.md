# 贵阳小学生测评服务平台 - 功能需求文档

## 1. 项目概述

### 1.1 项目背景
贵阳小学生测评服务平台是基于数字化时代背景下，为贵阳市教育局开发的专业学科能力测评认证系统。本系统旨在构建全市统一的学科能力测评认证平台，为小学生提供标准化、数字化的学习测评服务，帮助教育部门科学评估学生学习水平，支持教育决策。

### 1.2 系统定位
- **目标用户**: 贵阳市小学生（6-12岁）、教师、教育管理人员、家长
- **应用场景**: 学科能力等级测评、模拟测评练习、个性化学习、能力认证
- **技术架构**: 三层架构（测评系统+成长学习资源系统+认证系统），支持多终端访问

### 1.3 核心价值
- **标准化测评**: 建立全市统一的学科能力测评标准体系
- **个性化学习**: 提供丰富的学习资源和模拟测评平台
- **权威性认证**: 颁发具有公信力的能力等级证书
- **数据化分析**: 实现多维度数据统计与精准分析
- **智能化管理**: 提升测评效率与管理水平
- **激励机制**: 基于八角行为分析法的成就系统激发学习动力

#### 4.3.5 活动生命周期管理 ✅ **已实现**

**活动状态流转**:
```
草稿(draft) → 已发布(published) → 进行中(ongoing) → 已结束(finished)
     ↓              ↓
   删除          取消发布 → 草稿 → 删除
```

**活动类型与操作规则**:

| 活动类型 | 创建 | 发布 | 取消发布 | 删除 | 删除方式 |
|---------|------|------|----------|------|----------|
| 练习(practice) | ✅ | ✅ | ✅ 无人参与时可取消 | ✅ 仅草稿状态 | 硬删除（级联删除相关数据） |
| 测评(assessment) | ✅ | ✅ | ⚠️ 无人报名时可取消 | ✅ 仅草稿状态 | 硬删除（级联删除相关数据） |

**删除规则说明**:
- **只有草稿状态的活动可以删除**: 已发布的活动必须先取消发布才能删除
- **硬删除机制**: 删除活动时会级联删除所有相关数据：
  - `activity_questions` - 活动题目关联
  - `student_activities` - 学生参与记录
  - `answers` - 学生答案（通过 student_activities 级联）
- **数据不可恢复**: 删除操作不可逆，系统会提示用户确认

**取消发布规则**:
- **练习活动**: 只要没有学生参与（`student_activities` 表无记录），即可取消发布
- **测评活动**: 发布后如有学生报名，则不能取消发布
